# Development Dependencies Installation
1. `forge`
    ```bash
    curl -L https://foundry.paradigm.xyz | bash
    foundryup
    ```
2. `lcov`/`genhtml`
    ```bash
    sudo apt install lcov
    ```
3. `solhint`
    ```bash
    npm install -g solhint
    ```
4. `slither`
    ```bash
    pipx install slither-analyzer
    ```
4. `aderyn` (Recommended)
    ```bash
    npm install -g @cyfrin/aderyn
    ```
